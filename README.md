# 🌟 小嵐網站

🎉 歡迎來到 **小嵐網站 (XiaoLan Website)** GitHub 專案！  
這是一個專注於 **前端網頁設計** 的個人作品，使用 HTML、CSS、JavaScript 和 Tailwind CSS 打造現代化網站。

## 🚀 專案特色
- ✅ **HTML + CSS + JS 完整架構**
- ✅ **Tailwind CSS** 美化版型
- ✅ **RWD 響應式設計**，適用桌機與手機
- ✅ **部落格風格首頁**，展示文章列表
- ✅ **清晰的 UI 排版 + 側邊欄設計**
